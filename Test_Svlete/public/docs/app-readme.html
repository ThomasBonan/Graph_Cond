Graph_Cond - aide rapide

Ce document recap les informations essentielles pour exploiter l application en mode editeur ou configurateur, sur site comme en environnement de test.

01. Mode editeur

  • Connexion obligatoire pour creer, modifier ou supprimer un schema.
  • Nommer le schema puis utiliser Enregistrer / Mettre a jour.
  • Dupliquer pour iterer et Supprimer pour retirer un schema.
  • Historique via Annuler / Retablir et Restaurer un brouillon local.

02. Mode configurateur

  • Lecture seule pour controler les regles appliquees.
  • Selection via la liste deroulante, filtres par groupe et gamme.
  • Import / export JSON pour partager un schema.
  • Reinitialiser pour revenir a l etat neutre.

03. Sauvegardes et brouillons

  • Chaque enregistrement stocke le schema dans SQLite.
  • Brouillon local conserve dans localStorage apres rafraichissement.
  • Notifications (toasts) confirment import, export et erreurs API.
  • Pensez a nettoyer les brouillons obsoletes.

04. Notifications et aide

  • Les toasts (notifications en haut a droite) confirment import/export, sauvegardes et erreurs.
  • Le bouton ? ouvre ce guide et un acces au README utilisateur.
  • La recherche accepte noms de noeuds, groupes ou sous-groupes.
  • Les filtres groupe/gamme reduisent l affichage pour se concentrer sur un sous-ensemble.

05. Raccourcis clavier

  • Ctrl ou Cmd + S - Sauvegarder le schema courant.
  • Ctrl ou Cmd + F - Focus sur la recherche / ouverture du menu.
  • Escape - Fermer menu, aide ou connexion.
Besoin d aide supplementaire ?
Utilisez le bouton ? en bas a droite de l interface pour ouvrir l aide contextuelle directement dans l application.
